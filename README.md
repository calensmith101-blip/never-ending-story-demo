# Never Ending Story demo

This is the safe website trial build.

- Story generation returns local sample content only.
- No story requests are sent to a backend or external model provider.
- Browser persistence is disabled; profile edits and story counters reset on refresh.
- Mobile packaging, backend routes, service credentials, signing material, dependencies, lockfiles, screenshots, and build output are excluded from the demo package.

Run locally with:

```bash
npm install
npm run dev
```
